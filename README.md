# RecordTalk

# 這是一個紀錄LINE聊天的系統
# 可以透過他操控帳號聊天並紀錄內容
# 請至record.py輸入你的Token
# 請將程式中LINE("")的雙掛號內資料換成你的token
# 若第一次使用token可以留空

# 執行方法(僅簡介Android的方法)

# 請安裝以下app:
# 1.Termux
# 2.Quick Edit
# 3.Hackers keyboard

# 於Termux中逐行貼上以下內容:
# pkg install python -y
# pkg install git -y
# pkg install nano -y
# pip install thrift==0.11.0
# pip install requests
# pip install rsa
# git clone https://github.com/KazumiLine/RecordTalk
# cd RecordTalk
# ls
